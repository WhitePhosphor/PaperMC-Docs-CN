## 要求

:::tip

Paper 至少需要 **Java 21** 才能运行，[下载和安装非常简单](/misc/java-install)。

:::

| Paper 版本       | 推荐的 Java 版本 |
|----------------|-------------|
| 1.8 to 1.11    | Java 8      |
| 1.12 to 1.16.4 | Java 11     |
| 1.16.5         | Java 16     |
| 1.17.1-1.18.1+ | Java 21     |

## 下载 Paper

Paper 从我们[网站的下载页面](https://papermc.io/downloads)
直接提供可运行的服务器 JAR 文件。

点击构建编号即可下载文件。

## 运行服务器

要运行服务器，您需要创建一个启动脚本，
或者在终端中运行一个命令。

您可以使用我们的[启动脚本生成器](/misc/tools/start-script-gen)生成启动脚本。
您也可以在那里获取运行服务器的优化终端命令。

如果您只是需要一个简短的命令：
```bash
java -Xms4G -Xmx4G -jar paper.jar --nogui
```
请确保您已将终端导航到服务器的目录，
并将 `paper.jar` 替换为您下载的 JAR 文件的名称。

您可以通过更改 `Xmx` 和 `Xms` 参数中的数字来设置 RAM 的数量。
`--nogui` 会禁用原版的图形界面，这样在使用命令行时就不会出现双重界面。

要配置您的服务器，请查看[全局配置](/paper/reference/global-configuration)
和[世界配置](/paper/reference/world-configuration)页面。

## 迁移到 Paper

### 从原版迁移

从原版迁移非常简单，但有一些区别，特别是在世界保存方面。
Paper（以及 CraftBukkit 和 Spigot）会将每个世界的各个维度（下界、末地等）分别保存到独立的世界文件夹中。

Paper 会自动处理这种转换，无需额外考虑。

### 从 CraftBukkit 或 Spigot 迁移

Paper 是 CraftBukkit 和 Spigot 的直接替代品，您无需进行任何更改。
